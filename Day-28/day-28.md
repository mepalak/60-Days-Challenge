# 🏥 Day 28 – Hospital Admission Readiness Simulator

## 📌 Overview

On **Day 28** of the **60 Days of Claude AI Challenge**, I built a **Hospital Admission Readiness Simulator** using Claude AI. This interactive web application simulates the hospital admission workflow, allowing users to evaluate patient readiness, verify required documentation, assess medical risk, and make informed admission decisions through a realistic healthcare scenario.

The project demonstrates how AI can be used to rapidly build domain-specific applications with intuitive user interfaces and structured decision-making processes.

---

## 🎯 Objectives

- Simulate a real-world hospital admission process.
- Assess patient admission readiness based on multiple criteria.
- Validate required medical documents and patient information.
- Calculate admission readiness scores.
- Provide instant recommendations for admission decisions.
- Enhance prompt engineering skills using Claude AI.

## ⚙️ How It Works

1. Start a new patient admission.
2. Enter patient details and medical information.
3. Verify identity and required documents.
4. Complete medical readiness assessment.
5. Review risk indicators.
6. Generate the admission readiness score.
7. Receive an admission recommendation.
8. Restart with a new patient scenario.
