# Day 7 - Installing and Exploring Claude Counter Extension

## 📌 Overview

On Day 7 of my AI learning journey, I explored how browser extensions can enhance productivity while working with AI tools. I installed and configured the **Claude Counter extension** in Chrome/Chromium to track and analyze my interactions with Claude.

This activity helped me understand how AI tools can be extended using lightweight browser-based solutions.

---

## 🎯 Objectives

* Learn how to install a custom Chrome extension using Developer Mode.
* Understand the structure of a browser extension.
* Explore how AI productivity tools can improve workflow.
* Gain hands-on experience with Chromium-based browser extensions.

---

## 🛠️ Steps Followed

1. Downloaded the `claude-counter-0.4.2.zip` extension file.
2. Opened Chrome Extensions using:

```
chrome://extensions
```

3. Enabled **Developer Mode**.
4. Extracted the ZIP file.
5. Loaded the unpacked extension using the **Load unpacked** option.
6. Verified that the Claude Counter extension was successfully installed.
7. Tested the extension while interacting with Claude.
