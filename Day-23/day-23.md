# Day 23 – Customer & MVP Blueprint Generator

## 📌 Overview
On **Day 23** of the **#60DayClaudeChallenge**, I built an AI-powered **Customer & MVP Blueprint Generator** using Claude. This prompt acts like an experienced **Startup Product Manager and Customer Research Expert**, helping founders validate their startup ideas and generate a structured product strategy.

The AI first determines whether a **Startup Validation Report** already exists. Based on the available information, it extracts or collects essential startup details and creates a concise, PDF-ready report covering customer research, MVP planning, prioritization, pricing, risks, and execution strategy.

---

## 🎯 Objective
To transform a startup idea into a practical customer validation and MVP roadmap by identifying customer needs, prioritizing features, and defining actionable next steps.

---

## 🛠️ What I Built
A Claude prompt that:

- Checks for an existing Startup Validation Report.
- Extracts startup information if available.
- Collects startup details through guided questions if not available.
- Generates a comprehensive Customer & MVP Blueprint.
- Produces a concise report suitable for PDF export.

---

## 📋 Inputs Required

If a Startup Validation Report exists:
- Startup Idea
- Problem Statement
- Target Customers
- Competitors
- Validation Insights

Otherwise, Claude asks for:

1. Startup Idea
2. Problem Being Solved
3. Target Customers
4. Existing Validation
5. Target Market / Country
