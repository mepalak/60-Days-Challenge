# Day 8 – Prompt Chaining

## 📌 Objective
Learn how Prompt Chaining works by breaking a complex task into multiple smaller prompts. Instead of asking AI to complete everything in a single prompt, each prompt builds upon the previous response to produce better and more accurate results.

---

## 🧠 What is Prompt Chaining?

Prompt Chaining is a technique where multiple prompts are connected together. The output of one prompt becomes the input for the next prompt.

Example Flow:

Idea Generation
⬇
Research
⬇
Content Draft
⬇
Review
⬇
Final Output

This approach helps AI generate more structured, reliable, and high-quality responses.

---

## 🔗 Prompt Chain Used

### Prompt 1
Generate five project ideas for a beginner AI Engineer.

### Prompt 2
Select the best project idea and create a detailed implementation plan.

### Prompt 3
Generate the folder structure and required technologies.

### Prompt 4
Write the README file for the project.

### Prompt 5
Review and improve the README by making it more professional.

---

## ✅ Key Learnings

- Broke complex tasks into smaller steps.
- Improved response quality using sequential prompts.
- Learned how context is maintained across prompts.
- Reduced AI mistakes by refining outputs gradually.
- Created structured documentation efficiently.

---

## 💡 Observations

- Smaller prompts provide more accurate results.
- Each prompt improves the previous response.
- Prompt Chaining is useful for software development and documentation.
- It makes AI responses easier to review and edit.
