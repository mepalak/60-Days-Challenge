## 🧠 What is Prompt Chaining?

Prompt Chaining is a technique where a large problem is divided into multiple smaller prompts.

Instead of asking the AI to solve everything at once, each prompt focuses on one specific task. The output from one prompt becomes the input for the next, resulting in more accurate and organized responses.

---

## 🔄 Workflow

```
Problem
   │
   ▼
Prompt 1 → Understand the task
   │
   ▼
Prompt 2 → Generate ideas
   │
   ▼
Prompt 3 → Improve the solution
   │
   ▼
Prompt 4 → Review & Optimize
   │
   ▼
Final Output
```

---

## 💬 Example Prompt Chain

### Prompt 1
Explain the concept of Machine Learning in simple language.

### Output
A beginner-friendly explanation of Machine Learning.

---

### Prompt 2

Generate interview questions based on the explanation.

### Output

A list of beginner and intermediate interview questions.

---

### Prompt 3

Provide concise answers for each interview question.

### Output

Well-structured answers for quick revision.

---

### Prompt 4

Convert everything into Markdown notes.

### Output

A complete study guide ready for documentation.
