# 🚀 Day 26 — Prior Authorization Workflow Simulator

## 📌 Challenge Overview

For **Day 26** of the **60 Days Claude Challenge**, I built an interactive **Prior Authorization Workflow Simulator** using Claude AI.

The goal of this project was to understand how a healthcare prior authorization process works by simulating a patient's journey through different workflow stages — from initiating a request to payer decision and potential appeal.

Using Claude, I generated a complete HTML application that transformed a complex healthcare workflow into an interactive learning experience.

---

## 🎯 Objective

The objective of this challenge was to:

- Understand the Prior Authorization workflow
- Simulate different patient scenarios
- Track the patient's journey through the process
- Assemble required documentation
- Submit requests to the payer
- Handle approval, pending, denial, and appeal outcomes
- Understand delays and elapsed days in the workflow
